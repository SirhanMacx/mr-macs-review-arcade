ℹ️ Privacy & How It Works

All data shown here lives on this device only. Mr. Mac's Review Arcade is a static site served from GitHub Pages — there is no backend, no analytics, no student-identifying transmission. Each browser keeps its own profile in localStorage. When you click "Reset This Browser", that profile is cleared and the device starts fresh.

For roster-style class management, use the Live Multiplayer tool above and click "📋 Save as Class" in the lobby. Multiple saved rosters appear in the panel below — no accounts, no cloud sync, no logins. Everything stays on this device.
