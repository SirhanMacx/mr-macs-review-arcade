Support status

This page is the Review Arcade support candidate for App Store preparation. It is ready to publish as static support content, but public release still requires live URL verification, final App Store privacy answers, and legal/privacy review before monetization.

Local data

Use Settings in Review Arcade to export local data or reset local profiles, shards, rankings, rooms, and locker inventory on the device.

Accounts

Current prototype logins are local profiles only. There is no production remote account, password, classroom roster sync, or cross-device sync system yet.

Purchases

Purchases are disabled. Store Preview products are fixed, non-random planning entries until App Store Connect products, receipt validation, server entitlements, and legal/privacy review are complete.

Support and privacy contact

Use Review Arcade Support for support, privacy questions, leaderboard removal, bug reports, and purchase-readiness questions. Do not include student names, school ids, passwords, phone numbers, or private classroom records in a request.

Deletion and removal requests

Local device data can be exported or reset in Settings. For leaderboard removal, open a support request with the public handle, game or Review Board, approximate date, and score so the matching public row can be reviewed.
